Media

Image

JPEG / JPG

JPEG, short for "Joint Photographic Expert Group" is a grid-based image format using lossy compresison. It is one of the most popular image formats and supported on almost any browser / OS.

Pros
  • Supported almost anywhere
  • Small file sizes
Cons
  • Low image quality
PNG

PNG, short for "Portable Network Graphic" is a grid-based image format using lossless compression. It supports transparent pixels.

Pros
  • Supported almost anywhere
  • High image quality
  • Transparency support
Cons
  • Larger file sizes
GIF

A GIF (Graphical Interchange Format) is a lossless format for image files that supports both animated and static images. It was the standard for 8-bit colour images on the internet until PNG became a viable alternative.

Pros
  • There are 256 colors in Graphic Interchange Format.
  • Graphic Interchange Format is perfect for logos.
  • It is ideal for small animations.d
Cons
  • It uses a bigger file size compared to the PNG format

Video

MP4

MPEG-4 Part 14 or MP4 is a digital multimedia container format most commonly used to store video and audio, but it can also be used to store other data such as subtitles and still images.

Pros
  • Good image quality at low data rates
  • Supports 3D objects
Cons
  • Standard is still being designed
WebM

The WebM format was created with the distinct goal of being used on websites. Used together with the HTML5 specification, it creates videos of very small file sizes.

Pros
  • Open-source and roaylty-free format
  • Widely used for HTML5 and Chromecast
  • Greatly simplifies streaming
Cons
  • Not supported on a wide variety of mobile devices
MOV

QuickTime File Format is a computer file format used natively by the QuickTime framework.

Pros
  • Easily editable
  • Used in PCs and Macs
Cons
  • Mostly interchangeable within a QuickTime-only environment

Audio

MP3

The MP3 format is a popular audio file format which uses lossy compression.

Pros
  • Small file sizes
  • Good for storing audio on mobile devices
Cons
  • Low sound quality
WAV

WAV is the standard format in which all CDs are encoded. It does not use any compression.

Pros
  • Great sound quality
Cons
  • Huge file sizes
  • Poor metadata support (e.g. album artwork, artist, song title)
Ogg

OGG, short for "Ogg Vorbis", is a lossy, open-source alternative to MP3. This format is used in modern streaming services e.g. Spotify.

Pros
  • Open-source
  • Small file sizes
Cons
  • Low audio quality
